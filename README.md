# MDSAudobooks

* **Features**:
  * Search for the necessary books from the MDS catalog audiobooks 
  * Play books on the main screen or in the background.

___
* **Knowledges**:
  * Rest API
  * UIKIT
  * FileManager
  * AVKit
  * JSON
  * URLSession
  * Basic Animation 
  * Background playback
